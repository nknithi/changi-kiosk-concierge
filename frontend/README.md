# ChangiAI Kiosk - Frontend

This is the React frontend for the ChangiAI Kiosk prototype, developed for the Hipster technical assessment. 
It features a modern, glassmorphic UI, responsive layouts for kiosks and mobile screens, and integrates with the Node.js/Gemini backend.

## Tech Stack
- React 19
- Material UI (MUI) v6
- Axios for API communication

## Scripts

- `npm start` - Runs the app in development mode at http://localhost:3000
- `npm run build` - Builds the app for production deployment
